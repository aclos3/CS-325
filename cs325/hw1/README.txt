This is README.txt for explaining how to compile insertsort.py and mergesort.py

###For insertsort.py

1. Make the file be excutable on ssh
chmod +x insertsort.py

2. As the file is written as python 3, compile with this command on ssh
python3 insertsort.py

3. After few sentences, check the result txt file
cat insert.txt
or
vim insert.txt

###For mergesort.py

1. Make the file be excutable on ssh
chmod +x mergesort.py

2. As the file is written as python 3, compile with this command on ssh
python3 mergesort.py

3. After few sentences, check the result txt file
cat merge.txt
or
vim merge.txt

###For insertTime.py or mergeTime.py

1. Make the file be excutable on ssh
chmod +x insertTime.py or chmod +x mergeTime.py

2. As the file is wrriten as python 3, compile with this command on ssh
python3 insertTime.py or python3 mergeTime.py

3. check results !
