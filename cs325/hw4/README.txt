This is README.txt for explaining how to compile last_to_start.py

###if you have last_to_start.py file on your ssh directory

1. Make the file be excutable on ssh
chmod +x last_to_start.py

2. Before compile last_to_start.py, make sure the input file(act.txt) is existed on your directory

3. As the file is written as python 3, compile with this command on ssh
python3 last_to_start.py
