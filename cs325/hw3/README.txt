This is README.txt for explaining how to compile shopping.py

###if you have shopping.py file on your ssh path

1. Make the file be excutable on ssh
chmod +x shopping.py

2. As the file is written as python 3, compile with this command on ssh
python3 shopping.py

3. After printing few checking sentences, check the result txt file
cat results.txt
or
vim results.txt
